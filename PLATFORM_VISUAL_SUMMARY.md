## 🚀 Quick Start

### Initialize Services
```python
from app.services.data_mesh_service import get_data_mesh_service
from app.services.aiops_self_healing_service import get_aiops_service
from app.services.gitops_policy_service import get_gitops_service

# Get service instances
data_mesh = get_data_mesh_service()
aiops = get_aiops_service()
gitops = get_gitops_service()
```

### Use API
```bash
# Get platform overview
curl http://localhost:5000/api/v1/platform/overview

# Create data contract
curl -X POST http://localhost:5000/api/v1/platform/data-mesh/contracts \
  -H "Content-Type: application/json" \
  -d '{"domain": "user_management", "name": "User Contract", ...}'

# Get AIOps metrics
curl http://localhost:5000/api/v1/platform/aiops/metrics
```
