# خريطة شاملة للمستودع

يوفر الملف `app/tooling/repository_map.py` وسيلة سريعة لاستخراج صورة كاملة عن بنية المشروع الحالية. الهدف هو مساعدة المطورين الجدد على فهم جميع الملفات والمجلدات، وتحديد المسارات الميتة أو المتقادمة التي يجب تنظيفها.

## كيفية الاستخدام
1. من جذر المشروع شغّل:
   ```bash
   python app/tooling/repository_map.py
   ```
   سيُطبع تمثيل JSON يتضمن كل المجلدات والملفات مع الحجم.
2. لتضمين الملفات المخفية (مثل ملفات الضبط) نفّذ:
   ```bash
   python -m app.tooling.repository_map --include-hidden
   ```
3. يمكن تمرير `--max-depth` لتقليل مستوى التفاصيل أثناء الاستكشاف الأولي.

## فوائد الخريطة
- تحديد الملفات الميتة أو المتكررة لتبسيط الشيفرة.
- الحصول على رؤية موحّدة تغطي كامل المشروع بدل التركيز على ملف أو اثنين.
- توجيه المساهمين الجدد إلى أماكن الشيفرة الفعلية بسرعة.
