# 🎨 الدليل المرئي - شرح بالرسومات

## 🏗️ بنية المشروع بالرسم

```
        🌐 الإنترنت
           │
           ↓
    ┌──────────────┐
    │  المستخدم 👤  │
    └──────┬───────┘
           │
           ↓ (يرسل طلب)
    ┌──────────────────────┐
    │   🚪 main.py         │  ← الباب الرئيسي
    │   (نقطة الدخول)      │
    └──────┬───────────────┘
           │
           ↓
    ┌──────────────────────┐
    │   🧠 kernel.py       │  ← المخ المدبر
    │   (يوجه الطلبات)     │
    └──────┬───────────────┘
           │
           ├─────────────────┐
           │                 │
           ↓                 ↓
    ┌─────────────┐   ┌─────────────┐
    │ 🔒 security │   │ 🍽️ services │
    │  (الأمان)   │   │  (الخدمات)  │
    └─────┬───────┘   └─────┬───────┘
          │                 │
          ↓                 ↓
    ┌─────────────────────────┐
    │   📚 models.py          │  ← قاعدة البيانات
    │   (تخزين المعلومات)     │
    └─────────────────────────┘
```

    │                                           │

    │  ↓                                        │
    │  ┌─────────────────────────────────────┐ │

## 🔄 كيف تتواصل الأجزاء؟

```
    المستخدم
       │
       ↓ (طلب)
    ┌─────┐
    │ API │  ← واجهة البرمجة (النادل)
    └──┬──┘
       │
       ↓ (يوجه الطلب)
    ┌─────────┐
    │ Service │  ← الخدمة (الطباخ)
    └────┬────┘
         │
         ↓ (يجلب البيانات)
    ┌──────────┐
    │ Database │  ← قاعدة البيانات (المخزن)
    └────┬─────┘
         │
         ↓ (يرجع البيانات)
    ┌─────────┐
    │ Service │
    └────┬────┘
         │
         ↓ (يرجع النتيجة)
    ┌─────┐
    │ API │
    └──┬──┘
       │
       ↓ (رد)
    المستخدم
```
